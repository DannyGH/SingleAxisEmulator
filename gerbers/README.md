These Gerber files were created in EAGLE PCB using the [Sparkfun CAM file](https://learn.sparkfun.com/tutorials/using-eagle-board-layout/generating-gerbers)   

Here are the layers:

| Layer            | Ext. |
|:----------------:|:----:|
|Bottom Copper     | GBL  |
|Bottom Soldermask | GBS  |
|Top Copper        | GTL  |
|Top Silkscreen    | GTO  |
|Top Soldermask    | GTS  |
|Drill File        | TXT  |
